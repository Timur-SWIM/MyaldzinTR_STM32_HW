## Описание  
В данном репозитории представлены задачи, выполненные в рамках курса «Проектирование микропроцессорных систем».

## Структура репозитория  

├─ asm_dz1/ ← Задание 1: преобразование значения напряжения в код ЦАП на ассемблере

└─ README.md ← Этот файл

---

## Требования / Среда разработки  
- Микроконтроллер семейства STM32F103RBT6 
- Среда разработки: например, STM32CubeIDE или аналогичная, поддерживающая STM32.  
- Компилятор: arm-none-eabi-gc  
  
---

## Запуск и сборка  
1. Откройте проект в STM32CubeIDE и импортируйте туда как CMake-проект.  
2. Подключите отладчик, загрузите прошивку.  
5. При необходимости смотрите вывод через UART - Virtual COM Port.
